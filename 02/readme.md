# Exercice : Procedurale & OOP
Objectif : Créer un petit programme qui permet de gérer des livres (
    ajout, 
    suppression, 
    recherche, 
    affichage).
Fonctionnalités :
- Ajouter un livre (titre, auteur, année, genre)
- Supprimer un livre par titre
- Rechercher un livre
- Afficher tous les livres
